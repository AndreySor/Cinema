Инструкция по установке

Скачайте и установите Tomcat не ниже версии 9.0.43
Установите PostgreSQL и убедитесь в наличие соединения и активного пользователя
Подключитесь к БД PostgreSQL и выполните скрипты из папки cinema/src/main/resources/sql в следующем порядке schema.sql - data.sql
В файле application.properties пропишите необходимые для подключения к БД настройки.
Соберите проект командой mvn clean install
Перейдите в папку с установленным Tomcat
Перенесите .war архив проекта из папки target в папку webapps Tomcat
Запустите Tomcat (для MacOs исполнить catalina.sh из папки bin)
Пункт 5-8 можно исключить, используя IntelliJ IDEA
Откройте Edit configuration
Вверху слева + и выберите Tomcat local
Укажите в configure папку, куда был установлен Tomcat, выберите для деплоя war:exploded и задайте контекст /FWA
Все готово для использования

Как использовать приложение:
/admin/panel/halls - страница создания зала
/admin/panel/films - страница создания фильма
/admin/panel/sessions - страница создания сеанса на фильм в конкретном зале на конкретную дату
